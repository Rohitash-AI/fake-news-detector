# 🗞️ Fake News Detector

A Machine Learning web application that detects whether a news article is Real or Fake with 98.66% accuracy. In today's world, misinformation spreads faster than truth. This project tackles that problem using Natural Language Processing and Machine Learning. The user simply pastes any news headline or article, and the app instantly predicts whether it is Real or Fake along with a confidence percentage. The model was trained on 44,898 real world news articles, achieving 98.66% accuracy with 0.99 precision and recall. Input text is cleaned using NLTK, converted to numerical features using TF-IDF Vectorization, and classified using Logistic Regression — all running in real time through a Streamlit web interface.
